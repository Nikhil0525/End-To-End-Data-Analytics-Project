# End-to-End Data Analytics Project

## 📊 Overview
This project focuses on analyzing a sales dataset to extract meaningful business insights using SQL, Excel, and Power BI.

## 🛠️ Tools & Technologies
- SQL
- Excel
- Power BI

## 📌 Project Steps
1. Data Cleaning using Excel
2. Data Analysis using SQL
3. Data Visualization using Power BI

## 📊 Key Analysis
- Total Sales and Profit calculation
- Region-wise performance
- Category-wise sales analysis
- Top performing products

## 📈 Insights
- West region generated highest sales
- Technology category has highest profit margin
- Certain products contribute to majority of revenue
